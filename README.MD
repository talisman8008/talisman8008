## Featured Project — FirstMerge

A web platform and Chrome extension that helps first-time open source contributors find quality GitHub issues and verify their pull requests *before* submitting — catching avoidable mistakes that get PRs rejected, rather than letting contributors find out after the fact.

- **Live:** [firstmerge.app](https://www.firstmerge.app/)
- **Stack:** React, Node.js/Express, Supabase, Gemini AI, Chrome MV3 (Manifest V3)
- Browser extension analyzes PR diffs directly on `github.com/compare/` before submission
- Issue context resolved via automatic `#123` pattern matching against repo issues
- Secure auth bridge between the web app and extension using Supabase's `onAuthStateChange`, with a background service worker handling silent token refresh
- Hardened through a full security pass: SSRF mitigation, CORS lockdown, rate limiting, RLS policies, and prompt-injection sanitization on AI inputs
- Built and shipped for HackVerse 2026
